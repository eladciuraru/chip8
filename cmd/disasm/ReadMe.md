# DISASM
Utility program to disassemble CHIP8 compiled binaries.


# Usage
```shell
$ build\disasm.exe
  -b uint
        The base address of the first instruction (shorthand)
  -base uint
        The base address of the first instruction
  -f string
        Format string for each line in the disassembly, read docs for more - not implemented (shorthand)
  -format string
        Format string for each line in the disassembly, read docs for more - not implemented
  -l    Generate labels for control flows - not implemented (shorthand)
  -labels
        Generate labels for control flows - not implemented
  -p string
        Path to file contains CHIP-8 compiled instructions (shorthand)
  -path string
        Path to file contains CHIP-8 compiled instructions

```
